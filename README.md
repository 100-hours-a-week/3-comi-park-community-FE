```

cd kbt3-comi-community-fe
npm run start
```

### 디렉터리 구조

- `api/`
    - 백엔드 서버로 API 요청 보내는 자바스크립트 파일 존재

- `component/`
    - 다른 곳에서 활용되는 컴포넌트의 자바스크립트 및 CSS 파일 존재
    - 자바스크립트는 HTML 동적 생성 및 이벤트 등록, 공통 함수 정의

- `utils/`
    - 유틸리티 자바스크립트 파일 존재
